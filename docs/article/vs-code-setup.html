misc

howto

  1. create folder x ; cd into it
  2. uv init --no-workspace
    • we do not need a workspace for a single project
  3. create:
    1. .vscode/
    2. settings.json
    3. extensions.json
    4. .gitignore
    5. src/
    6. mv hello.py to src/main.py
    7. __init__.py
    8. tests/
    9. touch tests/test_main.py
    10. $ uv add --dev pytest
